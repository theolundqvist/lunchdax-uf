Vårt företag

Lunchdax UF är ett nystartat UF-företag som utvecklar en applikation för ios och android som kallas för "Lunchdax". Vi strävar efter kvalitet, funktionalitet, professionalism och ett lyckat UF-år.

Vår produkt

"Lunchdax" skapades för att göra det så lätt som möjligt för dig att se vad som finns att äta i ditt område. Allt för att du enkelt ska kunna hitta vad du vill äta, när du vill äta.

Vår tjänst

Via vår app kan restauranger enkelt få bra och kontinuerlig marknadsföring och flöde till verksamheten eftersom det blir lätt för användare att hitta nya ställen att äta på.

Support

Om du stöter på problem eller har några frågor kan du kontakta oss via Messenger eller Mail.
